# 🌦 Weather Pro App

A modern and responsive weather application that provides real-time weather data with a clean UI and dynamic background changes based on weather conditions.

##  Features

- 📍 Get weather using **current location (Geolocation API)**
- 🔍 Search weather by city name
- 🌡 Displays temperature, weather condition, humidity, and wind speed
- 🎨 Dynamic background changes based on weather
- ⚡ Fast and responsive UI
- 🧠 Handles errors (invalid city / permission denied)

## Tech Stack

- HTML5
- CSS3 (Glassmorphism UI + Gradients)
- JavaScript (ES6)
- Fetch API
- Geolocation API

##  How It Works

- Fetches real-time weather data from an external API
- Uses JavaScript async/await for data handling
- Updates UI dynamically based on user input or location

##  Future Improvements
- 🌍 5-day weather forecast
- 📊 Weather charts (temperature trends)
- 📱 Improved mobile responsiveness
- 🎨 Weather icons & animations
